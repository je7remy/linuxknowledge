### 7- ¿Cuál es la versión mínima de Python requerida para usar FastAPI?

_Seleccione solamente LA MEJOR respuesta_

- **A. Python 3.6** (Respuesta correcta)
    
- B. Python 2.7 
    
- C. Python 3.4
    

---

> La respuesta correcta es 'A'.
> 
> FastAPI requiere al menos Python 3.6 para funcionar.

_**Nota:** El requisito de la versión 3.6 se debe a que fue la versión que introdujo los **Type Hints** (pistas de tipo), una característica fundamental sobre la que está construido FastAPI._


### 8- ¿Qué es un Framework?

_Seleccione solamente LA MEJOR respuesta_

- A. Un conjunto de reglas gramaticales para escribir código en diferentes lenguajes.
    
- B. Un programa que traduce código fuente a lenguaje máquina para su ejecución.
    
- **C. Un entorno estructurado que proporciona herramientas y componentes reutilizables para facilitar el desarrollo de aplicaciones.** (Seleccionada)
    

---

> Esta respuesta es correcta.
> 
> Un framework es un entorno estructurado que ofrece componentes reutilizables, librerías y convenciones que ayudan a los desarrolladores a construir aplicaciones de forma más rápida y organizada. En lugar de empezar desde cero, se aprovechan funcionalidades ya definidas, lo que mejora la eficiencia y la mantenibilidad del código.
