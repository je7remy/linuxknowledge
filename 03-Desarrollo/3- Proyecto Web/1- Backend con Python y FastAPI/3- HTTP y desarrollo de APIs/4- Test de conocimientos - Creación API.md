
### 1. ¿Qué es CORS en el contexto de APIs?

- **A)** Controla acceso entre dominios ✅
    
- **B)** Comprime las respuestas
    
- **C)** Convierte JSON en HTML
    

> **Justificación:** El CORS es el mecanismo que controla el acceso de recursos desde otros dominios.

- **B)** BaseModel ✅
    
- **C)** Schema

### 3. ¿Qué estructura devuelve FastAPI por defecto al retornar objetos?

- **A)** XML
    
- **B)** HTML

### 4. ¿Cuál es la convención para nombrar los endpoints que representan colecciones de recursos?

- **A)** Usar verbos de acción
    
- **B)** Usar sustantivos en plural ✅
    
- **C)** Usar sustantivos en singular
    

> **Justificación:** La práctica común en REST es usar sustantivos en plural para referirse a una colección de recursos, como `/productos` o `/articulos`.

- **C)** Un archivo JSON
