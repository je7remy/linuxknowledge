Nuevas Tecnologías

🚀 Backend (La Lógica)
- Lenguaje: Python
- Framework: FastAPI
- Pegamento (ORM): SQLAlchemy (para conectar tu lógica con la base de datos)

🎨 Frontend (Lo Visual)
- Base: HTML y JavaScript
- Estilos: Tailwind CSS (tu alternativa moderna a Bootstrap)

🗄️ Base de Datos (Los Datos)
- Motor: PostgreSQL (la versión profesional de SQL que reemplaza a SQLite)

🔒 Despliegue (El Servidor)
- Entorno: Servidor Local / Intranet (cumpliendo el requisito de privacidad de tu tesis)
- Método sugerido: Docker (para empaquetar y correr tu aplicación + base de datos en ese servidor local)
