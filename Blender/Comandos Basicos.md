1. Mover un objecto sobre si mismo = presionando el botón medio del mouse.
2. Mover un objecto en el espacio = presionando (shift) + el botón medio del mouse.
3. Mover objecto de lugar = seleccionando la herramienta mover + la letra G.
4. Rotar objecto de lugar = seleccionando la herramienta rotar + la letra R.
5. Escalar objecto de lugar = seleccionando la herramienta escalar + la letra S.
6. Abrir menú ADD (añadir) = presionando (shift) + A.
7. Cambiar tamaño del objecto (desde cualquier eje (x, y, z)) = primero seleccionamos la parte del objecto a mover, segundo seleccionamos la herramienta mover y por ultimo presionamos G + el eje a modificar, (también podemos hacerlo manualmente seleccionando las flechitas azul = eje z, verde = eje y, rojo = eje x ).
8. Insert (inserta cara internas en una cara seleccionada) = presionando la tecla I.
9. Seleccionar todas las caras = (shift) + (Alt).
10. Escalar sin perder la forma del objecto seleccionado = (Alt) + S. (simplemente podemos arrastrar el circulo blanco que nos aparece al darle click a la herramienta escalar).
11. Crear selección en un objecto = Posicionamos el cursor donde queremos la selección (Ctrl) + R.
12. Para cambiar de lugar una selección establecida = (shift) + (Alt) + click.
13. Zoom = Ctrl + la ruedita del mouse.
14. Extruir = E.
15. Seleccionar objecto = A.
16. Biselar = (Ctrl) + B. (se puede ralentizar la biselacion presionando (shift))
17. Deseleccionar = doble A.
18. Unir vértices asimétricos = con shift seleccionamos los vértices de donde hasta donde lo queremos y presionando j crea los vértices faltantes.
19. Herramienta cuchillo = K. 
20. Activar menú delete = X.
21. Arrastre de selección = B.
22. Opciones de vértices = alt + M.
23. Crear Aristas = se seleccionan 2 bordes donde queremos crear la cara apretando shift soltamos y presionemos F. 
24. Seleccionar objecto que esta dentro de otro objecto = L.
25. Duplicar objecto = (shift) + (D).
26. Limpiar posición del objecto = (Ctrl) + A, todas las transformaciones.
27. Unir objectos = con los objectos seleccionados presionamos (Ctrl) + j (no se pueden realizar animaciones a objectos individuales que se hayan unido de esta manera).
28. Emparentar objectos = con los objectos seleccionados presionamos (Ctrl) + P, mantener transformación.
29. Cambiar vista de la cámara =  (Ctrl) + (Alt) + 0. (si no funciona, entra en la pestaña editar, preferencias, entrada y active la opción de emular teclado numérico.)
30. Invertir selección = (Ctrl) + I.
31. Moverte alrededor de un objecto = seleccionar el objecto y presionar la tecla punto.
32. Abrir Menú Contextual = después de hacer una selección dar click derecho. 
33. Abrir Menú UV = después de hacer una selección dar click en la letra U.
34. Seleccionar en edición UV posicionamos el cursor y presionamos L.
35. Ocultar objecto Temporalmente = seleccionamos y presionamos H.
36. Aparecer objecto ocultado temporalmente = Alt + H.
37. Agrandar selección =  (Ctrl) + (+).
38. Tolbar = T.
39. Menú de transformación = n.
40. Moverse aun lugar especifico del espacio = Alt + botón medio del mouse.
41. Cargar nuevo archivo = Ctrl + n.
42. Menú de objecto nuevo = f9
43. Cambiar la posición del 3d Cursor = Shift + Click
44. Poner objecto en el punto de origen = Alt + G 
45. Activar menú Snap = (shift) + (S).
46. Poner el 3d cursor en el punto de origen Shift + S, (y muchas funciones chingonas), es un pay menú la ñ y el . también despliegan el suyo.
47. Deshacer = Ctrl + r.
48. Alternar tipos de selección = w.
49. Seleccionar = letra c.
50. Seleccionar con la herramienta lazo = Ctrl + click derecho.
51. Renderizar = F12
